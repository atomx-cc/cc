    南德

    1992.12《南德画报》序 #

    一九九二年十二月

    “江山代有才人出，各领风骚数百年”。西方在衰落，东方在复兴。这是历史的逻辑。

    东方文化的整体综合思维方式正在取代西方文化的局部分析思维方式而成为当代学术思想的主流；太平洋西海岸经济的高速增长与西方世界的经济萧条形成鲜明对比；汉文化经济圈的崛起将是21世纪世界经济发展的主旋律。可以预料，西方价值观念体系的瑰宝与东方传统文化的精华相结合必将构造出人类经济发展及社会进步的最佳秩序。这是辩证法的预言。

    历史正按照辩证法走着自己的路。中华民族千载难逢的复兴机会已经到来，2000年正是历史的关节点。我们——横跨两个世纪的历史幸运儿——没有理由徘徊。让我们抓住这个历史的机遇，在中华民族经济复兴的潮流中去建功立业!

    后记 #

    “五百年一个轮回，五百年必有王者兴。”这是古人的观点。

    “西方文化擅长分析，东方文化擅长综合。”这是今人的总结。

    “而今是知识爆炸的时代，任何个人都无法与每天千变万化的知识体系抗衡，因此西方文化的分析思维方式正在让位于东方文化的综合思维方式，中华民族伟大复兴的时代已经到来！”这是牟其中总裁对时代特征的高度概括。

    本文是牟其中总裁对时代特征的最初阐释。此后，“中美俄经济大三角”理论逐渐成形，南德业务的全球战略开始布局。
